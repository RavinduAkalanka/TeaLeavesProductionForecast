About

TeaSense helps you predict tea leaf production using smart machine learning models. Get accurate forecasts to improve your tea farming decisions and boost your harvest.

TeaSense is an AI-based forecasting system designed to help tea estate owners, managers, and supervisors in Sri Lanka’s Ratnapura District. It replaces unreliable manual yield estimates with accurate, data-driven predictions using machine learning. By analyzing historical yields, weather, and soil data, TeaSense helps improve harvest planning, reduce waste and fraud, and support more efficient and sustainable tea production.
